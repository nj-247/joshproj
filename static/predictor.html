The Making of this Project

Initially, I set out to build a simple web app to predict coronary heart disease risk using machine learning. However, the project evolved into a full-stack application where users can input health information and instantly receive a risk score, visualized interactively and paired with personalized recommendations. I learned how to integrate machine learning models with web technologies, as well as incvorporating data visualization libraries like Vega-Lite and D3.js to make the web app more engaging and informative. The project also taught me the importance of user experience design, as I focused on creating an intuitive interface that guides users through the diagnostic process. The final product is a comprehensive tool that not only predicts risk but also educates users about their health, making it a valuable resource for anyone concerned about coronary heart disease.

The project unfolded in five key phases:

  1. Data Collection: Sourced the Cleveland Heart Disease dataset.
  2. Model Training: Preprocessed data and trained a machine learning model.
  3. Frontend Development: Built an interactive web interface using HTML, CSS, and JavaScript.
  4. Visualization: Integrated Vega-Lite for dynamic risk visualization.
  5. Testing & Deployment: Validated the app's functionality and deployed it for public access.
